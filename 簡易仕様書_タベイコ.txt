## アプリケーション機能
### 機能一覧
スライド機能：メインページで日本の都市を表示することで日本のサイトの感じを反映。
レストラン検索：Geolocation APIとのホットペッパー グルメサーチAPIを使用して
　　　　　　　　現在位置周辺にある飲食店を検索する。
　　　　　　　　詳細条件を追加してフィルタリングされた検索ができる。
レストラン情報取得：ホットペッパー グルメサーチAPIを使用して飲食店の詳細情報を取得し、
　　　　　　　　　　Google Map APIを通じて飲食店の位置を表示する。

### 画面一覧
メイン画面：範囲を指定して検索。
　　　　　　スライド表示。
　　　　　　現在のユーザーの位置から一番近い店を5つ表示する。
一覧画面：検索結果の飲食店一覧表示する。
飲食店詳細情報表示画面：飲食店詳細情報表示、地図連携、レビューを投稿を表示する。

### 使用しているAPI,SDK,ライブラリなど
- ホットペッパーグルメサーチAPI
- Geolocation API
- Pagination ライブラリ
- bxSlider ライブラリ
- Google Map API
